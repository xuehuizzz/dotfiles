### 使用
```bash
# 可视模式-----------------------
dd 删除光标所在的那一行
d 删除选中的部分
u 撤销上一步
y 复制到粘贴板
p 粘贴自粘贴板上的内容
```


### 快捷键

- 缩进和取消缩进
  - <mark>tab,   shift + tab</mark> (NORMAL, VISUAL)
- 打开终端
  - <mark>ctrl + j</mark>
- 全选
  - <mark>ctrl + a</mark> (NORMAL, VISUAL, INSERT)
- 注释
  - <mark>ctrl + /</mark> (NORMAL, VISUAL)
